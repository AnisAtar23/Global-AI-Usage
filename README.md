# 🌍 Global AI Adoption in Education (2015–2026)

> **A Comprehensive Global Performance Intelligence Report using Tableau, HTML, CSS, JavaScript, and Bootstrap**

---

## 📖 Project Overview

The **Global AI Adoption in Education (2015–2026)** project is an interactive Business Intelligence and Data Analytics solution that explores the adoption, usage, and growth of Artificial Intelligence in education across multiple countries and regions.

The project transforms raw educational data into meaningful insights using **Tableau dashboards**, **interactive visualizations**, and **data storytelling**. It helps educators, researchers, policymakers, and organizations understand global AI adoption trends and make informed decisions.
Website Link- https://global-ai-usage.vercel.app/index.html
---

## 🎯 Problem Statement

Artificial Intelligence is rapidly transforming the education sector worldwide. However, AI adoption varies significantly across different countries and regions. Educational institutions require reliable insights into AI adoption, teacher usage, student engagement, and regional growth patterns.

This project addresses these challenges by creating interactive dashboards and a Tableau Story that visualize AI adoption trends from **2015 to 2026**.

---

## 🎯 Objectives

- Analyze global AI adoption trends in education.
- Compare AI adoption across countries and regions.
- Monitor teacher and student AI usage.
- Identify high, medium, and low AI adoption levels.
- Visualize yearly AI growth trends.
- Build interactive dashboards for data-driven decision-making.
- Present findings through Tableau Story.

# 📈 Dashboard 1 – AI Adoption Analysis

This dashboard focuses on global AI adoption trends.

### Features

- 🌍 Regional AI Adoption Map
- 📊 AI Adoption Level Breakdown

- 🌎 Regional Comparison
- 📌 Interactive Filters

### Key Insights
